# Images
- controlling the size of an image is  css : this can be done by using the width and the height properties in css.
- Aligning images using css: 
img.align-left {
float: left;
margin-right: 10px;}
img.align-right {
float: right;
margin-left: 10px;}
img.medium {
width: 250px;
height: 250px;} 
- Centering images Using CSS: 
Img.align-center { 
display: block; 
margin: 0px auto;} 
img.medium { 
width: 250px; 
height: 250px;}
